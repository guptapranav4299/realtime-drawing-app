# SocketIO drawing app 

## Technologies used
1)Express
2)NodeJs
3)P5js
4)Socket.io



## Getting started

Start using Node

```bash
# Install dependencies for server
npm install

# Run the server
node server
```

Start using Docker

```bash
# Building the image
docker build --tag socketiodrawing .

# Run the image in a container
docker run -d -p 3000:3000 socketiodrawing
```
